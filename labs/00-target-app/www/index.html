All cryptographic algorithms on this server are vulnerable to quantum computers!

This server uses algorithms that were considered secure in 2019 but will be broken by sufficiently powerful quantum computers:

  • RSA-2048: Vulnerable to Shor's algorithm
  • ECDHE (Elliptic Curve): Vulnerable to Shor's algorithm
  • AES-256: Partially weakened by Grover's algorithm (but still practical)

Next steps in this lab: You'll analyze these vulnerabilities and migrate to hybrid post-quantum cryptography.
